Routing Specification Pipeline

This workspace functions as a browser-native router. Because every tool lives in its own folder, searching matches identifiers and resolves slugs down to localized directories. Selecting custom exam criteria pre-wires specific aspect configurations in memory before passing control blocks to the standalone HTML5 Resizer Engine inside /tools/image-tools/resize-image/.
